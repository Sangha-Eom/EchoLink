- Eclipse 사용 시
상단 메뉴에서 Run -> Run Configurations... 로 이동합니다.
왼쪽 Java Application 목록에서 현재 실행 중인 설정을 선택합니다.
오른쪽 탭에서 Arguments 탭으로 이동합니다.

1. VM options 입력란이 나타나면 아래 내용을 그대로 복사하여 붙여넣습니다.
-Dspring.config.location=file:secret/application.yml

2. Working directory 섹션에서 Default 대신 Other: 를 선택합니다.
Workspace... 버튼을 클릭하여 Server 프로젝트를 선택하거나, File System... 버튼으로 직접 Server 폴더 경로를 지정합니다.
Apply 버튼을 눌러 저장한 후 다시 실행합니다.
