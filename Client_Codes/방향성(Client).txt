o클라이언트용 안드로이드 서버 제어 어플
IDE: Android Studio
Language: JAVA

-----
- 원격 전
    - 로그인
    - 서버(Desktop)과 연동
    - 설정
      ㄴ 해상도 지정
      ㄴ 비트레이트 조정(화질, 지연시간)
      ㄴ fps 상한선
    
- 원격 중
    - 설정
      ㄴ 버튼
        - 버튼에 특정 키 지정
        - 버튼 크기 조절
        - 버튼 위치 조절
      ㄴ 프리셋 생성
        ㄴ '버튼 생성' 후 키 지정, 배치한 위치한 키 배치 불러오기(1번, 2번, ...)
      ㄴ 마우스 이동 방식
        ㄴ 마우스 따라가기(화면을 터치한 곳에 마우스 이동)
        ㄴ 마우스 이동(화면을 드래그하여 마우스 이동)
      ㄴ 기본 제공 조이스틱
        ㄴ 가상 패드 등

    - 기능
      ㄴ 3손가락 클릭 - 키보드(안드로이드) 보여주기
      ㄴ 2손가락 클릭 - 우클릭
      ㄴ 1손가락 클릭 - 좌클릭
      ㄴ 드래그 - 마우스 컨트롤(마우스 이동 / 마우스 따라가기 선택 가능)

- 원격 후
  - 서버는 그대로 킨 상태로 '원격 전' 창으로 되돌아가기
